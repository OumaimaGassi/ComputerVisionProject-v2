Document Scanner - Input Dataset
=================================

This folder contains input images used for testing the document scanner pipeline.
Each image contains a document photographed from various angles and lighting 
conditions, with clear visible corners against a contrasting background.

Images:
-------
- sample_document_1.png : A4 paper document on a wooden desk (angled view)
- sample_receipt.png    : Receipt on a dark surface (narrow document)
- sample_bookpage.png   : Book page photographed from above

All images simulate realistic scanning conditions with perspective distortion,
different document sizes, and contrasting backgrounds.

Supported formats: .jpg, .jpeg, .png, .bmp, .tiff
